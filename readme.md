# Elham E2E Tests

## How to run the tests

1. Install the dependencies

```bash
yarn install
```
 or
```bash
npm install
```

2. Run the tests
run all test cases
```bash
yarn test 
```
run tests for login
```bash
yarn test-login 
```
run tests for groups
```bash
yarn test-groups 
```
run tests for catalog
```bash
yarn test-catalog 
```
run tests for profileprofile
```bash
yarn test-profile 
```
run tests for assesstance
```bash
yarn test-assesstance 
```
